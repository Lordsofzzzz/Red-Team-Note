## Red Team Phases (At a Glance)

1. **Reconnaissance** — Gather information with OSINT, Nmap, Gobuster, Shodan.
2. **Enumeration** — Identify hosts, services, and vulnerabilities.
3. **Initial Access** — Phishing, exploit kits, credential theft, web exploits.
4. **Privilege Escalation** — Achieve admin/root access.
5. **Lateral Movement** — Expand network foothold.
6. **Persistence** — Maintain long-term access.
7. **Exfiltration** — Steal data via covert channels.
8. **Evasion Techniques** — Bypass defenses, manipulate logs.
9. **Tools & Cheat Sheets** — Curated collection for efficient operations.
